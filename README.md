## Instalando Dependências

> Backend
```bash
cd server/ 
npm install
``` 
> Frontend
```bash
cd web/
npm install
``` 
## Executando aplicação

* Para rodar o back-end:

  ```
  cd server/ && npm start
  ```
* Para rodar o front-end:

  > Para o front-end funcionar corretamente é necessário que o back-end esteja rodando!

  ```
    cd web/ && npm run dev
  ```
